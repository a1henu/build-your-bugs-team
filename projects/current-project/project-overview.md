## 项目简介

基于大模型技术开发的托福/雅思作文Task2智能备考辅助系统，实现自动问答和模拟机器评分功能，为广大学子提供免费、高质量的备考辅助工具。

## 核心功能模块

### 1. 考法解析模块

- 题型定义查询
- 出题频率统计
- 答题逻辑指导
- 高频话题分析

### 2. 自动问答模块

- 基于RAG技术的知识库问答
- 多轮对话支持
- 备考规划指导
- 写作技巧查询

### 3. 模拟机器评分模块

- 多维度评分（任务回应度、组织连贯性、语言运用）
- 错误识别和反馈
- 针对性修改建议
- 分数变化跟踪
